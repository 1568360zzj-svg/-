# 暗域消消乐 - H5 消除游戏

## 项目简介
暗域消消乐是一款暗黑风格的 H5 三消游戏，在幽暗的世界中消除宝石、触发特殊效果、闯过关卡。

## 特性
- 🎮 **8×8 棋盘** 7 种元素宝石
- ✨ **特殊宝石系统** — 4连直线清除、5连彩虹炸弹、L/T形区域炸弹
- 🧊 **障碍系统** — 冰块(1-3层)、锁链、石头、泥沼、限时炸弹
- 🌀 **传送门** — 宝石下落时穿越空间
- 🎨 **4大区域主题** — 暗域入口/冰封深渊/熔岩之心/虚空深处
- 🎯 **多种关卡目标** — 分数/收集/指定类型/制造特殊/清除障碍
- ♾️ **无尽模式** — 无限挑战，冲最高分
- 📋 **20关预定义关卡** + 之后无限随机生成
- 💾 **本地存档** — 自动保存最高关卡和总分
- 🔊 **音效系统** — Web Audio API 程序化合成
- 🎆 **粒子特效 + 屏幕震动 + 闪白**
- 💡 **提示系统** — 4秒无操作高亮可交换位置
- ⭐ **关卡评价 + 过渡动画**

## 障碍类型
| 障碍 | 效果 | 出现关卡 |
|------|------|---------|
| 冰块 | 1-3层覆盖，相邻消除破冰 | Lv2+ |
| 锁链 | 锁住宝石不可移动 | Lv4+ |
| 石头 | 不可移动障碍，特殊宝石可碎 | Lv6+ |
| 传送门 | 宝石下落时传送 | Lv11+ |
| 泥沼 | 每回合25%概率蔓延 | Lv16+ |
| 限时炸弹 | 倒计时归零爆炸清除3x3 | Lv17+ |

## 运行
直接打开 `game.html` 即可。
